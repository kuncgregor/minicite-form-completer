## GUI mode

Run:

npm run gui

Open:

http://localhost:3000

### CLI mode
### 1) Install

Run in this folder:

npm install

### 2) Configure

Create `config.json` from `config.example.json` and fill your real values.

Set multiple emails in `emails` array. The script prepares one browser tab per email.

## 3) Run once (CLI)

npm run run-once
